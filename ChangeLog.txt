2025/3/28  Version 4.00
  最初のバージョン。

2025/3/29
  ShiftJIS のテキストの判定に失敗する事があったため、文字コード判定処理を修正。

2025/4/2
  置換時の入力・メニューを VZ と同じ順番に変更。
  置換時の対象文字列を反転するように修正。

2025/4/4  Version 4.01
  カーソル行のアンダーライン表示オプション(変数 UnderLine)を追加。
  オプションメニューの表示順を変更。
  制御文字の色指定(変数 col_ctrl)が動作するよう修正。
  タブ文字の色指定(変数 col_tab)を追加。
  正規表現で UTF-8 のマルチバイト文字を認識するように修正。
  ※LC_CTYPE に環境変数 LANG の値を設定しているだけなので、OS や環境設定によっ
    ては期待通りの動作をしない可能性があります。
    LC_CTYPE に設定する文字列は変数 Locale でも指定可能です。

2025/4/8  Version 4.02
  マルチバイト文字の後にタブがあると表示やカーソル位置がずれていたのを修正。
  NFD 合成文字に対応。濁点と半濁点のみ対応しています。

2025/4/14  Version 4.03
  256 色表示のコンソールに対応。設定ファイル term.xterm-256color を追加。
  ターミナル設定用の環境変数名を NE_TERM から N8_TERM に変更。
  編集中にターミナルのサイズが変更された場合でも [ESC]D で復旧するように修正。

2025/4/18  Version 4.04
  ファイルの履歴をフルパスで保持し、終了時に保存するように修正。
  履歴にあるファイルを開くと前回開いたカーソル位置に移動するように修正。
  最後に編集していたファイルを開くオプション -r を追加。
  ロックファイル名を n8lock に変更。

2025/4/28  Version 4.05
  曖昧文字幅対応のオプション(変数 Ambiguous)を追加。
  機能していなかったフリーカーソルの設定を削除。
  NFD 合成文字を選択すると表示がおかしくなっていたのを修正。
  ファイラーを VZ Editor に準拠した操作体系に変更。
  ファイラーでソート順を選択できるように修正。
  ファイラーで指定した拡張子のファイルのみ表示する機能を追加。
  ファイラーでメニューでディレクトリ選択するための環境変数 N8_PATH を追加。

2025/4/30  Version 4.06
  ファイラーのマスクをワイルドカードと正規表現で指定できるよう修正。

2025/5/8   Version 4.07
  分割モード、リードオンリーモード、ファイルの二重化機能を追加。
  ステータスバーのインジケータ表示対応。システム情報との切替オプションを追加。
  新規ファイルや検索文字列等はウインドウを表示して入力するよう修正。
  ファイルクローズやエディタ終了時の動作を VZ Editor と同等になるように修正。
  メニュー項目や色、ファイラーの色を DOS/V 版の VZ Editor に近くなるように修正。
  オプション設定メニュー表示キーを F12 に変更。
  全角スペース表示オプションを追加。
  検索文字列入力時に F12 押下で検索オプションメニューを表示する機能を追加。
  メニューや入力ウインドウの枠の文字を指定するオプションを追加。

2025/5/8   Version 4.08
  正規表現 OFF 設定で置換をすると失敗して場合によっては落ちていたのを修正。
  置換メニューの表示位置の調整。

2025/6/4   Version 4.0.9
  ファイルクローズでおかしな動作になる場合があったのを修正。
  ファイラー上部のディレクトリ表示でマスク指定も表示するように修正。
  バージョン表記方法を変更。
  deb と rpm ファイルの提供開始。

2025/7/28   Version 4.1.0
  C 編集モードを追加。
  ファイラーの拡張子の表示幅を指定するオプションを追加。
  ファイラーで現在選択中のファイル名を一番下の枠に表示するように修正。
  ファイルセーブ、クローズ時の動作指定オプションを追加。

2025/8/8  Version 4.1.1
  SVS/IVS 異体字セレクタがあった場合にカーソル位置がずれていたのを修正。
  コマンドラインでディレクトリを指定した場合はファイラーを開くように修正。
  ファイルを開いたディレクトリの履歴を保存し、ファイラーの Dir メニューで履歴
  を表示して選択すると移動できるように修正。

2026/5/8  Version 4.1.2
  文字コード BOM 付き UTF-16LE/UTF-16BE のファイルの読み書きに対応。
  UTF-8 の 4 バイト文字の入力や削除時の不具合を修正。
  ファイラーのファイルコピーや移動等の終了時にコピーや移動したファイル数を表示
するよう修正。
  ファイラー上のファイルサイズ表記を修正。
  書込不可のファイルはリードオンリーモードで開くよう修正。
  Windows 対応版を作成。

2026/5/12 Version 4.1.3
  Windows 版で再描画の際の画面サイズ取得が正しく取れていなかったのを修正。
  Windows 版で NFD 合成文字があると表示が乱れていたのを修正。
  画面分割時のスクロール処理を修正。

2026/5/18 Version 4.1.4
  拡張子に対応した指定キーワードの色表示機能を追加。
  検索時にマッチした文字列の色を変更するよう修正。
  ターミナルの画面サイズ変更時に自動で再描画するよう修正。
  挿入モードと上書きモードでカーソル形状を変更できるよう修正。
  ESC 押下時等に IME を自動オフするオプションを追加。
  Windows 版でマウスホイールによる操作に対応。

2026/5/28 Version 4.1.5
  キーワード色表示で正規表現を使えるように修正。
  C 編集モードで } を入力した際の不具合を修正。
  Windows 版でファイルマスクの指定を正規表現にすると落ちていたのを修正。

2026/6/8 Version 4.1.6
  C/C++ 形式のコメント /* */ や // の色表示に対応。
